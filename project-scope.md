## Problem Statement
I have been searching jobs and now since ATP does the resume scan, its hard to change the resume everytime to tailor it to the job opening.

## Solution
Build an AI powered solution, to optimize the resume to tailor it to the job opening. Input will be job opening or link to the job opening. 

## Features
 - Input to get the job opening.
 - Input to get the user resume in pdf or text file.
 - Find the gaps and add that into the potential solution.
 - Provide a final summary to the user
 - Option to publish it into PDF file or text file.
